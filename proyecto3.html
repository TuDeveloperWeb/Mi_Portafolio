Sistema de Help Desk

El sistema Help Desk se realizo con PHP nativo, con el proposito de poder gestionar y atender sus reclamos de los usuarios de manera eficiente.
Proporciona una plataforma donde los usuarios pueden enviar sus problemas, consultas o solicitudes de asistencia, y el equipo de soporte puede dar seguimiento y resolver cada caso de manera oportuna.

Nota: Este proyecto se encuentra actualmente en mantenimiento.
